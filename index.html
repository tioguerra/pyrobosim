PyRoboSim.

PyRoboSim is a simple robot simulator written in Python. This simulator is not designed for performance, our focus is on easy usage allowing for fast code development specially in educational scenarios.

Installation

To download the simulator, just download the latest development version from GitHub:

$ git clone https://github.com/tioguerra/pyrobosim.git

In order to run the simulator you need to make sure you have the necessary dependencies installed. We tried to keep dependencies at a minimum. In order to be able to run this simulator you will need:

  • A standard installation of Python 2.7.x with NumPy
  • Open Physics Engine compiled with Python bindings
  • VTK 5.x with its Python bindings

Obs.: Depending on your system it is likely that you will also need to install Cython and maybe some other dependencies.

To test if your system is ready do the following

$ python
> import ode
> import vtk

If you do not see any error message, then you are ready to go!

Usage

  1. Start the server in a terminal
$ python server.py 8888
  1. Start the client in another terminal:
$ python client.py 127.0.0.1 8888 8889
